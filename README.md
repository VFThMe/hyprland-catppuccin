## Other utilities
**Gammastep** - Add Night Light to the system
```bash
sudo pacman -S gammastep && gammastep -O 2800
```
**Hyprshot** - Hyprshot made screenshot via slurp and grim
```bash
sudo pacman -S hyprshot
```
